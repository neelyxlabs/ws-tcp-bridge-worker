ws-tcp-bridge client example

This page signs a bridge URL and opens a WebSocket. The bridge forwards raw bytes to the destination TCP host:port. Anything typed in the "Send" area is transmitted as bytes. This is a raw-TCP test tool — for SFTP, use the @neelyxlabs/sshclient-sftp-wasm library as the client.

(not connected)

Warning: pasting a production HMAC secret into a public web page is a security anti-pattern. This page exists for local testing against a disposable secret. Real consumers should sign URLs inside their own codebase, not copy secrets into random demo pages.
